## Installation

- clone repo

  ```
  git clone https://github.com/AdamuAbba/come_buy.git
  ```

- install dependencies

  ```
  yarn
  ```

- run android build `locally or check package.json scripts command chords`

  ```
  npx expo run:android
  ```
